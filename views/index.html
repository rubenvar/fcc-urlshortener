URL Shortener Microservice

This app is a url shortener:

  1. Write the url you want to shorten as a parameter, after /s/
  2. If you input a valid url, the app will show you a JSON response, with both the original url and the short one.
  3. If you visit the short url, or any other previously created, it will redirect you to the original stored url.

Cool, huh?

You can check it easily:

Go to this url and see the JSON output.

https://skinny-cub.glitch.me/s/https://rubenvara.io/

Or visit this short url to check the redirection:

http://skinny-cub.glitch.me/588776
